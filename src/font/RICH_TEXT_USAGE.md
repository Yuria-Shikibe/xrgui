# 墨言夕 (mo_yanxi) 富文本系统使用指南

富文本系统允许在字符串中嵌入特定的指令，实现对字体、颜色、大小、偏移、样式等的动态控制。本文档将详细说明如何使用该系统。

## 基本语法

富文本指令使用 `{}` 包裹，基本格式为：

- `{指令名:参数}`：带有参数的指令。
- `{指令名}`：不带参数的指令（常用于状态的还原或开关）。

**转义规则：**
- 如果需要在文本中显示实际的 `{` 或 `}`，请使用 `{{` 或 `}}` 进行转义。
- 系统同样支持 `\` 转义字符（例如 `\{`）。

## 属性修改器符号

部分指令（如颜色、大小、偏移）的参数支持通过前缀符号来决定修改方式：

- `=` 或 无前缀：**绝对设置**（覆盖当前值）。
- `+`：**相对增加**（在当前值基础上加上给定值）。
- `*`：**相对乘算**（在当前值基础上乘以给定值）。

## 支持的指令列表

### 1. 颜色 (Color)
指令名：`c`, `#`, `color`

- **设置颜色：** `{c:#FF0000}` 或 `{#:#00FF00}`
- **修改器：** `{c:*#808080}`
- **查表映射：** 如果设置了 `rich_text_look_up_table`，可以直接使用颜色名称，如 `{c:primary_color}`。
- **还原/恢复默认颜色：** `{/c}`, `{/#}`, `{/color}` 或者 `{c}`（无参数）。
- **特殊用法：** `{#}` 或参数里只有前缀加 `#` (例如 `{*#}`) 会被映射为恢复默认（Fallback）颜色。

### 2. 字体大小 (Size)
指令名：`s`, `sz`, `size`

支持一到两个浮点数参数。
- **绝对大小：** `{s:24}`（将长宽同时设为 24）或 `{s:24,30}`（X=24, Y=30）。
- **相对放大：** `{s:*1.5}` 或 `{s:*1.5,1.5}`（字号放大 1.5 倍）。
- **相对增减：** `{s:+2}` 或 `{s:+-2}`。
- **还原：** `{/s}`, `{/sz}`, `{/size}` 或 `{s}`。

### 3. 位置偏移 (Offset)
指令名：`off`

支持一到两个浮点数参数，用于调整文字位置。
- **绝对偏移：** `{off:10,20}`
- **相对偏移：** `{off:+5,-5}`
- **还原：** `{/off}` 或 `{off}`。

### 4. 字体 (Font)
指令名：`f`, `font`

- **指定字体：** `{f:Arial}`。如果有提供 `rich_text_look_up_table`，系统会优先查找字体别名表。
- **还原默认字体：** `{/f}`, `{/font}` 或 `{f}`。

### 5. 文本样式 (Style)

粗体、斜体、下划线属于开关类指令：

- **粗体 (Bold):** `{b}` 开启， `{/b}` 关闭。
- **斜体 (Italic):** `{i}` 开启， `{/i}` 关闭。
- **下划线 (Underline):** `{u}` 开启， `{/u}` 关闭。

### 6. 上下标 (Script)

- **上标 (Superscript):** `{^}` 开启。
- **下标 (Subscript):** `{_}` 开启。
- **恢复正常 (结束上下标):** `{-}`, `{/^}`, `{/_}`。

> 注意：启用上下标时，字体大小会自动缩小（乘以 0.6），并根据当前排版方向（横排或竖排）自动调整 Y 轴或 X 轴的偏移量。

### 7. 文本外框 (Wrap Frame)
指令名：`wrap`, `w`

- **矩形框：** `{w:b}`
- **圆角框：** `{w:r}`
- **关闭外框：** `{/w}`, `{/wrap}`

### 8. OpenType 特性 (Feature)
指令名：`ftr`

传递给 HarfBuzz 的特性字符串。
- **格式：** `{ftr:+liga}`（开启）、`{ftr:-liga}`（关闭）、`{ftr:salt=1}`（指定值）。
- **还原：** `{/ftr}` 或 `{ftr}`。

## 批量还原指令

除了各个指令自己的还原标签（如 `{/c}`）外，系统还提供批量快速还原机制：

- `{/count}`（正整数）：还原最近执行的 `count` 个指令，例如 `{/2}` 会撤销最近设置的 2 个格式指令。
- `{/连续斜杠}`：批量撤销所有指令，例如 `{//}` 会根据斜杠的数量撤回同样数量的格式指令，如果不指定数量或者斜杠足够多，可以起到重置的作用。

## 示例

```text
正常文本{c:#FF0000}红色文本{/c}恢复正常。
正常文本{#:#00FF00}绿色文本{/2}正常文本（批量撤销）。
{s:*1.5}{b}加粗且大 1.5 倍的文本{//}恢复所有格式。
H{_}2{-}O 和 E=mc{^}2{-} 的化学物理公式。
{off:+0,-10}这段文字整体向上浮动 10 像素。{/off}
```

## 解析模式说明

在创建 `tokenized_text` 时，可以指定不同的 `tokenize_tag`：
- `def`（默认）：应用转义、解析指令，并在文本中移除指令占位符。
- `raw`：完全将其作为普通字符串输入，不进行解析。
- `kep`：解析指令，但在内存结构中保留原指令字符，保持双向映射（主要用于需要与原始字符串保持位置映射关系的应用场景）。
